# Libran 테마 플러그인 제작 및 적용 가이드 (Theme Plugin Manual)

Libran은 사용자가 직접 터미널 외관을 커스터마이징하고 공유할 수 있도록 **선언적 TOML 기반의 테마 플러그인 기능**을 제공합니다. 이 매뉴얼을 따라 나만의 고유한 컬러 스키마와 레이아웃 배치, 텍스트 스타일, 그리고 극대화된 그래픽 디테일을 설계해 보세요.

---

## 1. 개요 및 동작 방식
Libran의 테마 플러그인은 동적 라이브러리 로딩 방식이 아닌, 안정성과 성능 최적화가 보장된 **선언적 설정 파일 방식**을 사용합니다.
사용자가 테마 정의 파일(`.toml`)을 특정 폴더에 저장하고 프로그램에 해당 테마명을 지정하면, Libran이 실행 중에 이를 실시간으로 로딩하여 UI 테마와 레이아웃, 스타일을 전환합니다.

*   **테마 저장 경로**: `~/.libran/themes/<테마이름>.toml`
    *   예: `forest` 테마를 개발한다면, `~/.libran/themes/forest.toml` 파일로 생성합니다.
*   **지원하는 색상 포맷**:
    1.  **Hex 코드**: `"#RRGGBB"` (예: `fg = "#C0C0C0"`)
    2.  **RGB 테이블**: `{ r = <0~255>, g = <0~255>, b = <0~255> }` (예: `fg = { r = 192, g = 192, b = 192 }`)

## 2. 사용 가능한 테마 인자 (21개 색상 슬롯)
테마 파일에는 아래의 키(인자)들을 지정하여 각 UI 요소의 색상을 제어할 수 있습니다. 지정하지 않은 키는 기본 베이스 테마(다크 모드)의 값을 상속받습니다.

### 배경 및 패널 표면
*   `[theme.bg]`
    *   `color` (색상): 전체 화면의 기본 배경색입니다.
    *   `force` (불린): `true`로 설정하면 터미널 본래의 배경색 설정을 무시하고 지정한 `color`를 강제로 채웁니다. `false`로 설정하면 터미널의 기본 배경 투명도가 유지됩니다.
*   `surface` (색상): 왼쪽 사이드바나 팝업 대화창의 밑바탕이 되는 표면색입니다. 메인 배경색(`bg`)보다 살짝 밝거나 어둡게 주어 깊이감을 표현할 수 있습니다.

### 기본 텍스트 및 정보 표시
*   `fg` (색상): 일반 텍스트, 문헌 리스트의 본문 글자 색상입니다.
*   `title` (색상): 대화창 헤더 타이틀 텍스트, 메인 섹션 제목의 색상입니다.
*   `meta` (색상): 저자명, 저널명, 연도 등 상세 메타데이터 정보 텍스트의 색상입니다.
*   `dim` (색상): 비활성화된 메뉴, 도움말 힌트 등 덜 강조되어야 하는 보조 텍스트 색상입니다.

### 강조 및 상태 제어
*   `accent_primary` (색상): 애플리케이션의 핵심 정체성이 되는 주 강조색입니다. 패널 테두리, 주 카테고리 헤더, UDC/분류 키 레이블 등에 사용됩니다.
*   `accent_secondary` (색상): 보조 강조색입니다. 활성화된 탭 타이틀, 브랜드 로고 텍스트 등에 쓰입니다.
*   `selected` (색상): 리스트나 테이블 등에서 현재 선택된 문헌 레코드를 시각적으로 식별하기 위한 텍스트 색상입니다.
*   `focus_bg` / `focus_fg` (색상): 현재 포커스가 위치한 입력 폼이나 버튼 영역의 배경 및 전경 글자색입니다.
*   `key` (색상): 도움말이나 하단 메뉴바에 표시되는 단축키 안내 문자(예: `Enter`, `Esc`, `e`)의 색상입니다.

### 콘텐츠 전용 요소
*   `divider` (색상): 각 패널을 나누는 얇은 경계선의 색상입니다.
*   `tag` (색상): 문헌에 지정된 키워드 태그의 텍스트 색상입니다.
*   `udc` (색상): UDC 분류 코드 텍스트 색상입니다.
*   `code` (색상): 코드 스타일의 텍스트나 인용 키 포맷의 텍스트 색상입니다.
*   `search_bg` (색상): 상단 검색 바에 포커스가 갔을 때 입력창의 배경색입니다.

### 알림 및 메시지 상태
*   `error` (색상): 에러 메시지나 실패 경고 알림 텍스트 색상입니다.
*   `warning` (색상): 중요 경고 사항 텍스트 색상입니다.
*   `success` (색상): 성공 메시지나 완료 처리 알림 텍스트 색상입니다.

## 3. 레이아웃 토폴로지 설정 (Layout Settings)
테마 파일 내에 `[layout]` 섹션을 구성하여 CUI 화면 구성의 물리적 배치를 제어할 수 있습니다.

*   `sidebar_position` (문자열): 왼쪽 사이드바의 배치 위치를 정의합니다.
    *   `"left"`: 화면 왼쪽에 배치 (기본값)
    *   `"right"`: 화면 오른쪽에 배치
*   `search_position` (문자열): 문헌 검색 입력창의 상하 배치를 정의합니다.
    *   `"top"`: 문헌 리스트 상단에 배치 (기본값)
    *   `"bottom"`: 문헌 리스트 하단에 배치

---

## 4. 텍스트 위젯 굵기/밑줄 스타일 오버라이드 (Style Modifiers)
테마 파일 내에 각 UI 요소의 텍스트 데코레이션(굵게, 밑줄)을 제어하는 속성을 설정할 수 있습니다.

*   `title_bold` (불린): 대화창 및 화면의 메인 타이틀을 굵게 표시할지 여부
*   `title_underline` (불린): 타이틀 텍스트 밑에 밑줄을 표시할지 여부
*   `header_bold` (불린): 각 패널의 섹션 헤더를 굵게 표시할지 여부
*   `header_underline` (불린): 섹션 헤더 밑에 밑줄을 표시할지 여부

---

## 5. 극한의 CUI 비주얼 커스터마이징 파라미터 (Extreme Customization Options)
Libran은 완벽한 터미널 테마 커스터마이징(이하 극대화된 커스텀 사양)을 위해 다음과 같은 세부 파라미터를 추가 제공합니다.

### A. 테두리 스타일 (`border`)
패널 및 팝업 대화창의 경계선 형태를 테마에 최적화된 유니코드 셋으로 설정합니다.
*   `border_type` (문자열): 테두리 모서리 스타일을 선택합니다.
    *   `"plain"`: 기본 실선 (`┌`, `┐`, `└`, `┘`)
    *   `"rounded"`: 둥근 모서리 선 (`╭`, `╮`, `╰`, `╯`) - 캐주얼/모던 터미널에 추천
    *   `"double"`: 이중 테두리 선 (`╔`, `╗`, `╚`, `╝`) - 레트로/중후한 매뉴얼풍 추천
    *   `"thick"`: 두꺼운 단선 테두리
    *   `"ascii"`: 구형 셸 호환 테두리 (`+`, `-`, `|`)
*   `[theme.borders]`
    *   `show_top` / `show_bottom` / `show_left` / `show_right` (불린): 각각 상/하/좌/우 테두리선의 렌더링 활성화 여부를 개별 설정합니다. (모두 끄고 구분선만 남겨 미니멀한 UI 연출 가능)

### B. UI 인터랙션 기호/글리프 (`[theme.glyphs]`)
UI 내부 조작에 사용되는 포인트 및 구분 기호를 유니코드 문자로 맵핑합니다.
*   `active_pointer` (문자열): 리스트 및 테이블에서 현재 포커싱된 항목을 가리키는 화살표 문자 (기본값: `"►"`)
*   `bullet_marker` (문자열): 대화창이나 도움말에서 하위 목록을 나타내는 불릿 문자 (기본값: `"▸"`)
*   `radio_checked` / `radio_unchecked` (문자열): 라디오 버튼 활성화/비활성화 기호 (기본값: `"●"`, `"○"`)

### C. 진행률 표시줄 스타일 (`[theme.progress]`)
독서율 및 다운로드율을 표기하는 가로 바의 글리프 문자셋을 지정합니다.
*   `filled` (문자열): 진행률 표시줄의 채워진 영역 문자 (기본값: `"━"`)
*   `unfilled` (문자열): 진행률 표시줄의 비어 있는 영역 문자 (기본값: `"─"`)

### D. 문헌 읽기 상태 마커 (`[theme.status]`)
문헌 리스트의 '안 읽음', '읽는 중', '다 읽음' 상태를 개별 글리프와 고유 색상으로 매핑하여 하이라이팅을 극대화합니다.
*   `unread_marker` / `reading_marker` / `read_marker` (문자열): 각 상태별 상태 표시 글리프 문자
*   `unread_color` / `reading_color` / `read_color` (색상 포맷): 각 상태별 표시 마커의 개별 색상

### E. 스크롤바 문자셋 (`[theme.scrollbar]`)
스크롤 가능한 창의 우측 스크롤 바를 사용자 정의 문자로 대체합니다.
*   `thumb` (문자열): 스크롤바 이동 손잡이 문자 (기본값: `"█"`)
*   `track` (문자열): 스크롤바 배경 트랙 문자 (기본값: `"░"`)

## 6. 테마 파일 예시 (extreme_forest.toml)
아래는 극대화된 커스터마이징 파라미터가 전부 도입된 테마 플러그인 정의 파일 예시입니다.

```toml
# 극대화된 포레스트(Extreme Forest) 테마 예시

[theme.bg]
color = "#121814"
force = true

[theme]
surface = "#1A231C"
fg = "#D0DCD0"
accent_primary = "#7EA385"
accent_secondary = "#BACD92"
dim = "#607364"
divider = "#2A3A2F"
selected = "#F5F5DC"
focus_bg = "#2A3D2E"
focus_fg = "#E6EFE6"
title = "#E6EFE6"
meta = "#859A89"
key = "#A8C3A0"
tag = "#BACD92"
udc = "#7EA385"
code = "#D2B48C"
search_bg = "#1F2E23"
success = "#7EA385"
warning = "#D2B48C"
error = "#CD5C5C"

# 텍스트 스타일 데코레이션 오버라이드
title_bold = true
title_underline = false
header_bold = true
header_underline = false

# 레이아웃 토폴로지 변경 설정
[layout]
sidebar_position = "right"
search_position = "bottom"

# 테두리 및 스크롤바 극대화 설정
[theme.border]
border_type = "rounded"

[theme.borders]
show_top = true
show_bottom = true
show_left = true
show_right = true

[theme.scrollbar]
thumb = "┃"
track = " "

# 기호 및 마커 정의
[theme.glyphs]
active_pointer = " ➜ "
bullet_marker = " • "
radio_checked = "▣"
radio_unchecked = "□"

# 진행바 커스텀
[theme.progress]
filled = "█"
unfilled = "░"

# 읽기 상태 마커 및 개별 하이라이트색 지정
[theme.status]
unread_marker = "○"
unread_color = "#607364"
reading_marker = "◑"
reading_color = "#D2B48C"
read_marker = "●"
read_color = "#7EA385"
```

## 7. 커스텀 테마 적용 방법
생성한 테마 플러그인을 소프트웨어에 적용하는 방법은 두 가지가 있습니다.

### 방법 A: 설정 화면(UI)에서 변경 (권장)
1.  Libran 프로그램을 실행합니다.
2.  `p` 키를 눌러 **설정(Appearance) 대화창**을 엽니다.
3.  방향키를 사용하여 첫 번째 항목인 `테마 이름`에 포커스를 둡니다.
4.  `Enter` 키를 눌러 입력 모드로 진입합니다.
5.  적용하려는 테마명(예: `forest`)을 입력한 뒤 `Enter` 키를 누르면, 설정이 즉시 저장되며 UI 테마 및 레이아웃이 실시간으로 전환됩니다.
6.  `Esc` 또는 `p` 키를 눌러 설정을 닫습니다.

### 방법 B: 설정 파일 직접 수정
1.  `~/.libran/config.toml` 파일을 텍스트 에디터로 엽니다.
2.  `theme_name` 인자를 찾아 원하는 테마 이름으로 설정합니다.
    ```toml
    theme_name = "forest"
    ```
3.  파일을 저장한 뒤 Libran을 재시작합니다.

## 8. 불가능한 커스터마이징 영역 (한계점)
Libran은 CUI(터미널) 애플리케이션의 성능 최적화 및 레이아웃 안정성을 위해 외관 변경 범위를 다음으로 제한합니다.

1.  **서체 종류(Font Family) 변경 불가**: 터미널 폰트는 에뮬레이터 설정에 고정되어 있으므로 프로그램에서 폰트를 바꿀 수 없습니다.
2.  **애니메이션 및 전환 효과 부재**: 터미널 특성상 페이드 인/아웃 등의 시각 효과는 지원되지 않습니다.
3.  **커스텀 그래픽 리소스 로드 불가**: 벡터 이미지(SVG)나 비트맵 이미지(PNG)를 아이콘 등으로 로드할 수 없으며, 모든 리소스는 유니코드 텍스트 문자로 한정됩니다.
